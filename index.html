2. Qual a diferença entre um site estático e dinâmico?

No site dinâmico os usuários podem interagir com caixas de texto, botões e etc, coisa que não é possível no site estático.

3. Para que os programadores utilizam o Github?

Usam para compartilhar códigos e/ou comita-los.

4. O que significa o acrônimo C.R.U.D?

Create, Read, Update, Delete.

5. Qual é a função da linguagem SQL?

Controlar o banco de dados.

6. O que é A.P.I?

Interface de Programação de Aplicação que facilita a comunicação e troca de informaçõea.

7. Qual é a função do CSS?

Mudar a aparência do texto e do fundo do site.

8. O que significa "responsividade" em páginas WEB?

Significa adaptar o tamanho das suas páginas para diferentes dispositovos.

9. Qual o papel do U.X. (User Experience)?

Garantir uma boa navegação de usuario.

10. O que é um wireframe em páginas WEB?

O esboço.
